# Is It Kitchen Safe?

A static reference site answering "Can you put X in Y?" questions for kitchen appliances.

## Overview
This project contains 1500+ static pages detailing whether everyday items are safe for the dishwasher, microwave, freezer, oven, dryer, air fryer, and refrigerator.

## Tech Stack
- **Framework**: Astro.js
- **Styling**: Tailwind CSS v4
- **Hosting**: Cloudflare Pages
- **Architecture**: MPA (Multi-Page Application)

## Running Locally

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
